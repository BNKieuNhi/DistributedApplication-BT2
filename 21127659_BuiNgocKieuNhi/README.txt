Bài tập 2 - Giao diện website quản lý bài báo
Sinh viên: Bùi Ngọc Kiều Nhi - MSSV: 21127659

Cách chạy:
1. Mở file index.php
2. Dựa trên tham số GET controller & action để load giao diện.
   Ví dụ: index.php?controller=user&action=login
3. Các giao diện trong thư mục /views như sau:
   - views/layout/ chứa header và footer chung
   - views/user/ chứa form đăng nhập
   - views/papers/ chứa trang thêm và xem chi tiết bài báo
   - views/authors/ chứa thông tin và chỉnh sửa profile

Sử dụng CSS, Bootstrap trong /assets/css và JS trong /assets/js
